ebu_tt_live.bindings.raw.CreateFromDocument(xml_text, fallback_namespace=None, location_base=None, default_namespace=None)[source]

Parse the given XML and use the document element to create a Python instance.

@param xml_text An XML document. This should be data (Python 2 str or Python 3 bytes), or a text (Python 2 unicode or Python 3 str) in the L{pyxb._InputEncoding} encoding.

@keyword fallback_namespace An absent L{pyxb.Namespace} instance to use for unqualified names when there is no default namespace in scope. If unspecified or C{None}, the namespace of the module containing this function will be used, if it is an absent namespace.

@keyword location_base: An object to be recorded as the base of all L{pyxb.utils.utility.Location} instances associated with events and objects handled by the parser. You might pass the URI from which the document was obtained.

@keyword default_namespace An alias for @c fallback_namespace used in PyXB 1.1.4 through 1.2.6. It behaved like a default namespace only for absent namespaces.
